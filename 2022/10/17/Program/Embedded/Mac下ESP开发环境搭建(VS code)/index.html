ESP开发环境搭建(VS Code)

  1. 下载并安装Visual Studio Code.

  2. 通过单击VS Code一侧的活动栏中的扩展图标，或使用快捷键 ⇧``⌘``X打开扩展视图。

  3. 使用任何相关关键字搜索扩展，例如 espressif、esp-idf、esp32、esp32s2 等。

  4. 安装扩展。

  5. 安装ESP-IDF依赖，如果使用 WSL2，则安装 WSL 文档中指定的所需包。

  6. （可选）按 F1 并键入 ESP-IDF：选择保存配置设置的位置，可以是用户设置、工作区设置或工作区文件夹设置。 请查看处理多个项目以获取更多信息。 默认为用户设置。
